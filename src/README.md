## Run locally

From the **repository root** (not inside `src/`):

```bash
npm install
npm run dev
```

Default dev server: [http://localhost:5173](http://localhost:5173) (Vite default).

```bash
npm run build
npm run preview
npm run lint
```

HTML shell metadata (`index.html`) includes author, keywords, and `rel="author"` for **Numan Usman** / **UFoundBEC**.
